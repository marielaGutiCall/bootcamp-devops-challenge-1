  1. Que es un sistema de control de versiones?

    [Un Sistema de Control de Versiones es una herramienta que ayuda a gestionar y hacer un seguimiento de los cambios que se realiza sobre el codigo en el transcurso del tiempo.]

  2. Cual es la diferencia entre git y github?

    [Git es un sistema de control de versiones distribuido nos permite hacer un seguimiento de los cambios que vamos realizando en el codigo, como Git guarda las diferentes versiones de un proyecto, podemos retroceder a un commit o etapa anterior y revertir los cambios que hayamos hecho. Por otra parte Github es una plataforma web que trabaja con git para almacenar en la nube nuestros respositorios locales para luego pasar a ser respositorios remotos, esto permite la colaboracion entre los miembros de los equipos]

  3. Cómo creo una rama en git?

    [Una rama puede ser creada con: git branch o git checkout -b con esto se crea la rama y nos mueve directamente a esta rama creada]
